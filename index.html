断線利用レールループ生成ツール

断線を使用したレールループを生成してくれます。(beta版)
使い方はこちら



使い方

目標値に希望の値を入力し、モードを選択して「生成」ボタンを押すと結果が表示されます。
目標値が大きくなるにつれ計算量も増えるため、最大表示数を適宜調整してください。

レール(断線)種類


s = 直線レール
d = 斜線レール
c = 曲線レール

周回モード

↓sG1 = ↓G1-直線レール
↓sG2 = ↓G2-直線レール
↓sG3 = ↓G3-直線レール
↑sG1 = ↑G1-直線レール
↑sG2 = ↑G2-直線レール
↑sG3 = ↑G3-直線レール

往復モード

sG1 = G1-直線レール
sG2 = G2-直線レール
sG3 = G3-直線レール

ソート、絞り込みについて

ソート、絞り込みは「生成」ボタンを押した後に使用可能になります。
- sとdの個数の差の絶対値が小さい順にソートします。
- cの個数が少ない順にソートします。
- 断線の個数の合計が少ない順にソートします。
- 断線が1種類以下の組み合わせに絞り込みます。
- ソート、絞り込み前の状態に戻します。
